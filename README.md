# NHentai API
## Description
This is an API to interact with the manga of the [NHentai](https://nhentai.net) web through python3. It will show the following characteristics:
+ Name
+ Code
+ ID
- Parodies
- Characters
- Tags
- Artists
- Groups
- Languages
- Categories
- Pages
